# Calculator

A clean, modern, and responsive calculator web app. Built as a beginner project for my developer portfolio. It supports digit input, the four basic operators, decimal points, clear, delete, and equals, with full keyboard support. The math is handled with a safe calculation function instead of eval().

## Features

- Digit buttons (0–9) and decimal point
- Add, subtract, multiply, divide
- Clear (C) and delete (backspace)
- Equals with correct calculations
- Full **keyboard support** (numbers, operators, Enter, Backspace, Escape)
- Responsive design with button hover effects
